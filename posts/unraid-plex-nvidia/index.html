    UnraidеҲқдҪ“йӘҢ

Unraid е®үиЈ… Nvidia й©ұеҠЁе№¶ејҖеҗҜ Plex зЎ¬и§Ј

еҸ‘иЎЁдәҺ 2022-12-10 Walineпјҡ йҳ…иҜ»ж¬Ўж•°пјҡ жң¬ж–Үеӯ—ж•°пјҡ 1.3k йҳ…иҜ»ж—¶й•ҝ ≈ 5 еҲҶй’ҹ

еүҚйқўи®°еҪ•дәҶдёҖдёӢжҲ‘еҲҡиЈ…еҘҪ UnraidпјҢеҲқе§ӢеҢ–й…ҚзҪ®пјҢеҸҠжҗӯе»әзЈҒзӣҳйҳөеҲ—зҡ„иҝҮзЁӢгҖӮдёҖдёӘ Nas иЈ…еҘҪпјҢе°‘дёҚдәҶиҰҒж•ҙдёҖдёӘе®¶еәӯеҪұйҹізі»з»ҹз”Ёиө·жқҘпјҢд»ҠеӨ©е°ұжқҘи®°еҪ•дёҖдёӢпјҢе®үиЈ… Plex е№¶ејҖеҗҜ Nvidia жҳҫеҚЎзЎ¬и§Јзҡ„иҝҮзЁӢгҖӮ

дёҖгҖҒеүҚжңҹиҜҙжҳҺ

йҰ–е…ҲжҲ‘жқҘи®ІдёҖдёӢжҲ‘зҡ„й…ҚзҪ®пјҢжҲ‘иҝҷиҫ№з”Ёзҡ„жҳҜд»Һдё»еҠӣжңәдёҠжҚўдёӢзҡ„ Ryzen 5 3600 й…ҚдёҖеј  Nvidia Quadra T600гҖӮ3600 жҳҜжІЎжңүж ёжҳҫзҡ„пјҢжҲ‘е°ұеҮҶеӨҮи®©иҝҷеј  T600 жқҘжүҝжӢ…иҪ¬з Ғзҡ„д»»еҠЎгҖӮ

дёәе•ҘйҖүжӢ© Nvidia Quadra T600 иҝҷдёӘеҚЎе‘ўпјҹ

йҰ–е…ҲиҝҷжҳҜдёҖеј  Quadra еҚЎпјҢжІЎжңүдёҖиҲ¬жҳҫеҚЎеҜ№ NVENC зҡ„йҷҗеҲ¶пјҢжҲ‘жҳҜдёҖдёӘжҮ’еҫ—жҠҳи…ҫзҡ„дәәпјҢйҖүжӢ© Quadra еҚЎе°ұзңҒдәҶеҫҲеӨҡдәӢгҖӮ

еҶҚдёҖдёӘпјҢеҠҹиҖ—еҫҲдҪҺпјҢжҳҫеӯҳе’ҢжҖ§иғҪд№ҹз¬ҰеҗҲжҲ‘зҡ„йңҖжұӮгҖӮжҲ‘ T600 иҝҷеј еҚЎпјҢ40W еҠҹиҖ—пјҢдёҚз”ЁйўқеӨ–дҫӣз”өпјҢжҸ’дёҠе°ұиғҪз”ЁпјҢиҝҳжңү 4GB жҳҫеӯҳпјҢзӣёжҜ”дёҠд»Ј P зі»еҲ—еҸӘжҸҗеҚҮдәҶ 10W еҠҹиҖ—пјҢжҖ§иғҪжҸҗеҚҮжҢәеӨ§гҖӮдёҚиҝҮжҜ•з«ҹж–°еҚЎпјҢд»·ж јиҝҳжҳҜдёҚеӨӘзҫҺдёҪгҖӮеҰӮжһңдёҖиҲ¬дҪҝз”ЁпјҢйҖү P400 иҝҷеј  30W зҡ„еҚЎпјҢд№ҹе°ұеҸҜд»ҘдәҶпјҢд»·ж јдҫҝе®ңеӨӘеӨҡгҖӮ

дәҢгҖҒж•ҷзЁӢ

иҜҙжҳҜж•ҷзЁӢпјҢе…¶е®һе°ұжҳҜжҲ‘ж“ҚдҪңзҡ„дёҖдёӘи®°еҪ•пјҢжҲ‘еңЁзҪ‘дёҠжҗңзҡ„еҘҪдәӣж•ҷзЁӢпјҢеҮ д№ҺйғҪжҳҜз”Ёзҡ„ Intel ж ёжҳҫзҡ„пјҢз”Ё N еҚЎзҡ„дёҚеӨҡпјҢи®Із©¶дёҖдёӘжҖ§д»·жҜ”пјҢеғҸжҲ‘иҝҷз§Қз”Ё Amd е№іеҸ°зҡ„е°ұжӣҙе°‘дәҶгҖӮд№ҹе°ұжҳҜжҲ‘жІЎз”Ё APUпјҢдёҚз„¶ Amd зҡ„ж ёжҳҫдј°и®Ўд№ҹжңүзӮ№иҰҒжҠҳи…ҫгҖӮ

еҸҜиғҪйңҖиҰҒз”ЁзӮ№йӯ”жі•пјҢдёҚз„¶дёӢиҪҪйҖҹеәҰе•Ҙзҡ„йғҪдјҡжҜ”иҫғж…ўгҖӮ

1. е®үиЈ…й©ұеҠЁ

еңЁеә”з”ЁдёӯеҝғпјҢжҗңзҙўnvidia driverпјҢе°ұиғҪзңӢеҲ°дёҖдёӘз”ұ ich777 еӨ§дҪ¬з»ҙжҠӨзҡ„ Nvidia-Driver жҸ’д»¶пјҢзӮ№еҮ»е®үиЈ…гҖӮ

е®үиЈ…е®ҢжҲҗеҗҺпјҢиҝӣе…Ҙи®ҫзҪ®пјҢзӮ№еҮ»иҝӣе…ҘдёӢж–№еҲҡеҲҡе®үиЈ…зҡ„еә”з”ЁгҖӮеә”з”ЁдјҡжӢүеҸ–жңҖж–°зҡ„й©ұеҠЁдҝЎжҒҜпјҢжүҖд»ҘдјҡжҜ”иҫғж…ўгҖӮ

йҖүжӢ©еҲҶж”ҜжҲ–иҖ…жҢҮе®ҡзҡ„й©ұеҠЁзүҲжң¬пјҢзӮ№еҮ» Update & Download жҢүй’®гҖӮиҝҷйҮҢжҲ‘йҖүжӢ© Production еҲҶж”ҜпјҢж„ҹи§үдјҡжҜ”иҫғзЁігҖӮ

зӯүеҫ…й©ұеҠЁдёӢиҪҪе®ҢжҲҗпјҢеј№зӘ—дҝЎжҒҜйҮҢдјҡжҳҫзӨәдёӢиҪҪе®ҢжҲҗпјҢйҮҚеҗҜд»Ҙе®үиЈ…ж–°зҡ„й©ұеҠЁпјҢеј№зӘ—дёӢж–№зҡ„жҢүй’®дјҡеҸҳдёәе®ҢжҲҗгҖӮжӯӨж—¶еҸҜд»ҘйҮҚеҗҜ UnraidпјҢж–°зҡ„й©ұеҠЁдјҡиҮӘеҠЁе®үиЈ…гҖӮеҚғдёҮдёҚиҰҒеңЁдёӯйҖ”е…ій—ӯеј№зӘ—пјҢиҝҷж ·е°ұжҗһдёҚжё…жҘҡзҠ¶жҖҒдәҶгҖӮ

иЈ…еҘҪеҗҺпјҢеҶҚж¬Ўиҝӣе…Ҙеә”з”ЁпјҢеә”з”ЁйҮҢе°ұдјҡжңүй©ұеҠЁдҝЎжҒҜдәҶгҖӮ

2. е®үиЈ… GPU зӣ‘жҺ§жҸ’д»¶пјҲйҖүпјү

дёәдәҶж–№дҫҝжҹҘзңӢ GPU зҠ¶жҖҒпјҢеҸҜд»ҘиҖғиҷ‘е®үиЈ…дёҖдёӘ GPU Statistics жҸ’д»¶гҖӮ

иЈ…еҘҪеҗҺпјҢе°ұеҸҜд»ҘеңЁд»ӘиЎЁзӣҳзңӢеҲ° GPU зҡ„иҙҹиҪҪжғ…еҶөгҖӮ

3. е®үиЈ… Plex

жҺҘдёӢжқҘе°ұжҳҜжҲ‘д»¬зҡ„дё»и§’дәҶгҖӮ

(1) йҖүжӢ©еә”з”Ё

иҝҷйҮҢжҲ‘йҖүжӢ©дәҶbinhex-plexpassиҝҷдёӘеә”з”ЁпјҢйҖүе…¶д»–зҡ„д№ҹеҸҜд»ҘпјҢиҝҷдёӘдё»иҰҒжҳҜжҜ”иҫғж–№дҫҝдёҖзӮ№пјҢдёҚе°‘еҸӮж•°е·Із»Ҹеё®дҪ еҲ—еҘҪдәҶпјҢеҸҜд»ҘзӣҙжҺҘй…ҚзҪ®гҖӮ

(2) дҝ®ж”№дё»иҰҒеҸӮж•°

зӮ№еҮ»е®үиЈ…иҝӣе…ҘеҸӮж•°и®ҫзҪ®пјҢдё»иҰҒжҳҜдҝ®ж”№иҝҷеҮ зӮ№гҖӮ

  • еҗҚз§°: Docker йЎөйқўеҗҺйқўжҳҫзӨәзҡ„еҗҚеӯ—пјҢеҸҜж”№еҸҜдёҚж”№
  • Host Path 2:(Container Path: /media): иҝҷдёӘжҳҜеӯҳеӮЁеҪұзүҮзҡ„и·Ҝеҫ„пјҢеҸҜиҮӘиЎҢдҝ®ж”№пјҢжүҖжңүзҡ„е…ұдә«йғҪеңЁиҝҷдёӘ/mnt/userдёӢйқўзңӢеҲ°
  • Key 3 (NVIDIA_VISIBLE_DEVICES): иҝҷдёӘжҳҜ GPU зҡ„ idпјҢеҸҜд»ҘеңЁ Nvidia Driver еә”з”ЁйҮҢжүҫеҲ°пјҢиҝҷдёӘжҢҮе®ҡдәҶ docker дҪҝз”Ёзҡ„жҳҫеҚЎгҖӮжіЁж„ҸпјҢдёҖеј жҳҫеҚЎеҸҜд»ҘеңЁеӨҡдёӘ docker еә”з”Ёдёӯе…ұз”ЁпјҢдҪҶдёҚиғҪеңЁ docker е’ҢиҷҡжӢҹжңәдёӯе…ұдә«гҖӮ

еңЁйўқеӨ–еҸӮж•°йҮҢеЎ«е…Ҙ--runtime=nvidiaпјҢжҢҮе®ҡдҪҝз”Ё Nvidia иҝҗиЎҢж—¶гҖӮ

(5) и®ҫзҪ®зҪ‘з»ңжЁЎејҸдёә bridge жЁЎејҸпјҲйҖүпјү

иҝҷдёӘеә”з”ЁжЁЎжқҝпјҢй»ҳи®Өз”Ёзҡ„жҳҜ host жЁЎејҸзҡ„зҪ‘з»ңпјҢеҰӮжһңдҪ дёҚе–ңж¬ўжғіиҰҒжҚўжҲҗ bridge жЁЎејҸпјҢе°ұзңӢдёӢиҝҷиҫ№пјҢеҗҰеҲҷи·іиҝҮиҝҷдёҖжӯҘгҖӮ

йҰ–е…ҲжҠҠзҪ‘з»ңжЁЎејҸеҲҮжҚўжҲҗ bridgeпјҢз„¶еҗҺж·»еҠ йңҖиҰҒжҡҙйңІзҡ„з«ҜеҸЈгҖӮ
