9. 合并冲突

当两个分支都对同一行进行了修改，git 便会产生冲突，并标记为未合并

此时将每个文件进行修改，确认最后的内容，使用 git add 方法标记为冲突已解决

在所有文件的冲突均已解决后，使用 commit 提交此次修改。
